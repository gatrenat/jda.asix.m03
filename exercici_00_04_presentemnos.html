Exercici 00_04. Presentem-nos¶

Context

  • Carpeta de lliurament: 00_04_presentemnos/

  • Continguts relacionats: Ritual Hola Món!

  • Com lliurar-lo: instruccions

Enunciat

En aquesta secció et proposo una senzilla modificació respecte el que hem vist al programa holamon.py. La seva realització et permetrà entendre una mica més com es fa servir l’eina de control de versions git.

Segueix les instruccions:

  1. Ves a la carpeta de treball del teu PC i actualitza canvis que puguin estar al repositori remot

    $ cd ~/introprg
$ git pull origin master

    En cas que et doni algun error, és possible que el teu espai de treball tingués canvis que no has registrat. Fes git status perquè git et recordi quins canvis vas fer, i fes el commit oportú. En cas de dubtes, avisa el teu docent que el tens desitjant ajudar-te.

  2. Crea la carpeta 00_03_presentemnos

    $ mkdir 00_03_presentemnos
$ cd 00_03_presentemnos

  3. Crea un nou fitxer empresento.py i escriu-hi codi tal que, en ser executat escrigui per pantalla el mateix que holamon.py, excepte que, en comptes de Hola Món!, sortirà un missatge similar al següent:

    Hola, sóc la Adelita, la futura millor desenvolupadora del Món!

    Per favor, si no et dius Adelita, canvia el nom. Pots modificar també la resta de la frase amb quelcom que sentis còmode.

  4. Aconsegueix que el programa pugui ser executat de la següent manera:

    $ ./empresento.py

  5. Afegeix els canvis al control de versions.

    Pots comprovar que git t’informa que hi ha un nou fitxer sense registrar, tot fent git status.

    Com ara hem creat un nou fitxer, ens caldrà informar a git que el volem gestionar:

    $ git add empresento.py

    Ara que el fitxer ja és conegut per git, demanem que afegeixi els canvis:

    $ git commit -am "Completat l'exercici 3"

  6. Finalment pugem els canvis a GitHub:

    $ git push origin master

    Com sempre, indica el teu password

A banda de l’exercici, et recomano que tu mateix et defineixis canvis que t’agradaria provar. Crea d’altres fitxers pels teus experiments, que pots mantenir sense problemes a aquesta carpeta. La màxima és: sense por. No pots trencar res!
